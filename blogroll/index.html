部落卷，也就是 Blogroll，是 blog（部落格）+ roll（清單），簡單來說就是我推薦的部落格。

這裡會包含我對這個部落格的描述、網址、及這個部落格的部落卷（也有人叫做部落卷卷）。

☕ 個人部落格
#

反正就是不是技術部落格的部落格。

Wiwi.Blog
#

連結：Wiwi.Blog

支持開源軟體，每天日更，反對大公司監控，也是「好和弦 NiceChord」的作者。

他也支持大家多寫部落格，然後用 RSS 互相訂閱。我也是從他知道「部落卷」這種東西的，他還有開一個 BlogBlog.Club (雖然才剛開始)。

部落卷：https://wiwi.blog/blogroll

Alex Hsu 徐小翔
#

網址：https://alexhsu.com/about

前 Meta 工程師，現在是獨立 APP 開發者，網站很有質感，雖然是工程師，但文章不是在講軟體方面的。

有很多哲學以及生活實驗，還有閱讀跟筆記，也會日更。

Ivon 的部落格
#

網址：https://ivonblog.com/

這是我最早開始看的部落格，最主要是 Linux 及其他開源軟體教學，也有開箱、刷機、人文等，也有一些 NSFW 內容。

更新很快，也有很多文章，不過這比較適合對電腦很有興趣的人。

部落卷：https://ivonblog.com/blogroll/

🧑‍💻 技術部落格
#

比較多在電腦程式方面的內容，有些也會有其他內容。

黑暗執行續
#

連結：blog.darkthread.net

一個資深工程師的技術部落格，有談論各種電腦相關的東西，頁面雖然有點老，但也會常常更新。

也有寫很多技術的內容，不過很多都是我不會的 .NET（C#），所以我都看不懂。

部落卷：無

Huli’s blog
#

技術的部落格：blog.huli.tw

比較生活的部落格：life.huli.tw

原本是前端工程師，後來變成資安工程師，目前住在日本，寫了很多程式教學，也有線上課程（我沒上過就是了）。

另一個生活的部落格則是什麼都寫，像是什麼理財、心得或是單純出去玩的紀錄。

部落卷：無

Code And Me
#

連結：blog.kyomind.tw

後端的 Python 工程師，也有筆記及寫作的心得，也有電子報。

我是看了他的「為什麼你「不需要」所謂的人生管理系統」這篇文章發現他的。

部落卷：無

其實我在寫描述時還真的常常不知道寫什麼，因為技術部落格又不一定全部技術，有些人的主題也很多元。

我會去看部落格的關於頁面，不過有時候真的很難寫，所以還是要自己看文章到底有沒有適合。
