是時候該離開 Chrome 了

本篇也投稿到WiWi的 2026 年 1 月份的 BlogBlog。

我原本使用 Google Chrome 作為瀏覽器，但是 Chrome 為了保障廣告收益（廣告佔據 Google 大約 70% 的收入），推出了名義上「保障使用者隱私」，實際上為了打壓廣告阻擋器的瀏覽器擴充規範。

在這個處處都充滿廣告跟追蹤器的時代，廣告阻擋已經對我而言是一個非常重要的功能，所以我決定要離開 Chrome。

發生了什麼事？
#

廣告阻擋器，也就是可以阻止追蹤器跟廣告的軟體，像是阻擋 YouTube 的廣告，加快載入速度（畢竟不用浪費網路載入廣告）。

在 Chrome 中可以用擴充功能達到，但是 Google 最近縮緊了擴充的權限，推出了 Manifest V3，禁止動態封鎖廣告（只能靜態宣告封鎖，不能執行邏輯，也不能注入遠端程式碼）。

實際上， AdGuard 這個強大的廣告阻擋器在 Google Play 也只能用閹割版（其實基本沒什麼功能，還只能在特定瀏覽器裡，要完整版必須到官網下載 APK ）。

Google 現在已經開始控制擴充功能。其實可以作到在使用者同意的情況下開放部分 API，但 Google 刻意選擇限制，控制了使用者的選擇。

目前，最大的開源廣告阻擋器 uBlock Origin 也已經在 Chrome 中被強制下架，無法使用。

我再也不可能回到 Chrome 了。

為什麼不選 Firefox
#

Firefox 已經變得和 Chrome 一樣。

首先， Firefox 官方移除了「我們不販售你的資料」這個承諾，並且更新了使用條款，允許 Mozilla 販賣 Firefox 用戶的使用者資料（雖然幾乎所有專有軟體都會這樣寫，但是已「隱私」為標榜的瀏覽器這樣做是很誇張）。

而且， Firefox 不僅沒有幫助使用者對抗追蹤器，反而還自動用了 Privacy-Preserving Attribution (PPA) 這個功能。簡單來說，就是把你的資料給廣告商，讓廣告商可以在「不用使用傳統追蹤」的方式下追蹤你（官方稱這個叫做歸因）。

Mozilla 甚至還買下了廣告公司，我知道這件事跟「隱私為主的網路」很矛盾，但這正是 Mozilla 在做的事。

註：PPA 這個功能在被告了之後已經關閉

Firefox 官方網站寫：

Firefox 由 Mozilla 於 2004 年開發，作為比 Internet Explorer 等瀏覽器更快、更隱私且可自訂的替代方案。如今，我們仍然是非營利組織，仍未被任何億萬富翁擁有，並持續努力讓網路——以及你花在網路上的時間——變得更好。

實際上，目前Mozilla 有 85％ 的收入來自 Google ，我無法想像一個用廣告賺錢的基金會，是要如何達到「乾淨網路」。

所以要去哪裡
#

或許 LibreWolf 會是個 Firefox 的替代品，但我個人是喜歡用 Brave 瀏覽器，因為內建著隱私及廣告阻擋。

而且 Brave 也是基於 Chromium （或許對於有些人來說這是缺點），不會遇到什麼使用太冷門的瀏覽器不支援的問題，套件也都可以用。

基本從 Chrome 換到 Brave 是很順的，密碼也可以直接匯入。甚至還有同步功能（雖然我只有一台電腦，不過可以當作備份），而且同步功能只需一組代碼，不需要任何帳號（這應該是 Web3 的概念）。

不過 Brave 最被批評的原因是大力推廣 Web3，但如果不主動使用，基本上不會受到影響。

而且 Brave 也是開源的，所以哪天你真的不滿意它的 Web3 及加密廣告，你也可以自己 Fork 自己的版本（已經有人在社群問過了）。

網路的價值
#

網路本身是自由的，但實際上開發一個軟體需要很大的力氣，而且如果要所有軟體都開源就幾乎要自架一大堆服務，而且真的技術難度很高。

有時候專有服務很方便，但是很多軟體真的不只那幾個，試著找找看別的，說不定會找到新歡。

我並不是個強求開源的人，我現在就是使用 Windows 11 寫這篇文章的。

我知道 Windows 很難用，但 Linux 真的太不穩定了（我重灌超級多次），或許我未來也會逃離 Windows 吧。
